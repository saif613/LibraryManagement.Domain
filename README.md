# 📚 Library Management System

A scalable library management system built using ASP.NET Core Web API, following Onion Architecture principles.

---

## 🚀 Features

- Manage books, categories, and users
- Borrowing and returning workflow
- Authentication & Authorization using ASP.NET Identity
- Clean and maintainable architecture
- Validation and error handling
- Logging and middleware support

---

## 🏗️ Architecture

This project follows **Onion Architecture**, structured into:

- **Domain Layer** → Core business logic and entities  
- **Application Layer** → Services, DTOs, and interfaces  
- **Infrastructure Layer** → Database, repositories, Identity  
- **Presentation Layer** → API controllers and middleware  

---

## 🛠️ Technologies Used

- ASP.NET Core Web API
- C#
- SQL Server
- Entity Framework Core
- LINQ
- AutoMapper
- Fluent Validation
- ASP.NET Identity
